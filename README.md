# waepic

Unofficial wrapper for WhatsApp Web API, built with Rust. This project is in an early stage.

## Acknowledgements

- [Baileys](https://github.com/WhiskeySockets/Baileys) - TypeScript implementation
- [grammers](https://codeberg.org/Lonami/grammers) - API design and reference
- [whatsmeow](https://github.com/tulir/whatsmeow) - Go implementation
- [whatsapp-rust](https://github.com/jlucaso1/whatsapp-rust) - wacore and waproto

## Contributing

We accept pull requests from our [Forjego instance](https://git.amanoteam.com/AmanoTeam/waepic) and [GitHub](https://github.com/AmanoTeam/waepic). Fork either repository, create a feature branch, and submit a pull request. Bug reports and feature requests are also welcome via the issue tracker.
